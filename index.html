Perfil

No es un portfolio clásico.

Soy diseñador multimedia y visual designer. Trabajo entre identidad visual, contenido audiovisual, diseño editorial, motion y experiencias digitales.

Mi background técnico en Unity, web y multimedia me permite entender sistemas interactivos, pero el foco principal de este portfolio es la dirección visual contemporánea.

Diseño con intención: jerarquía, ritmo, contraste, tono y coherencia visual.
Perfil híbrido: sensibilidad gráfica, cultura audiovisual y capacidad técnica para llevar ideas a producción.

Construyamos algo visual.

Disponible para proyectos de branding, audiovisual, motion, editorial y dirección visual para marcas, artistas y espacios culturales.
